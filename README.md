# Система для поиска образовательных организаций с аномальными результатами успеваемости

---

## Описание

Система разделена на 2 отдельных веб-интерфейса, которые можно запускать паралельно, однако входные данные для второго сервиса являются выходными для первого

Возможности, предоставляемые системой:
- Загрузка предварительно прокластеризованной и обработанной  выборке и разделение каждого кластера на 2 части по правилам, заданными пользователем
- Обучение двух регрессионных моделей `CatBoostRegressor` для каждого кластера (каждая модель для одной части) для прогнозирования уровня успеваемости
- Построение графиков для каждой регрессионной модели:
    - Точечного графика для предсказания и реальных данных
    - Частотного распределения для предсказаний и реальных данных
    - Графиков важности признаков на основе `SHAP`
- Скачивание данных для каждого кластера с предсказанным уровнем успеваемости и остатками (разностью между реальными данными и предсказаниями);
- Загрузка предобработанного кластера из прошлого пункта;
- Выбор признаков, на которых будут обучаться алгоритмы поиска аномалий;
- Возможность задания границ, в которых могут находиться оптимальные гиперпараметры для каждого алгоритма;
- Запуск процедуры поиска оптимальных гиперпараметров для каждого алгоритма
- Построение графиков для каждой модели поиска аномалий:
    - Двумерного графика норм и аномалий с помощью сжатия признакового пространства при помощи алгоритма `PCA`
    - Графика "выбросности" объектов
    - Графиков важности признаков на основе `SHAP`
- Возможность выбора любого числа алгоритмов и порогового значения для определения финальных аномалий;
- Возможность скачивания найденных финальных аномалий;

Интерактивный веб-интерфейс реализован на базе фреймворка `streamlit` и разделён на 2 части - одна часть прогнозирует успеваемость, другая ищет аномалии

---

## Стек технологий

- Язык прогнозирования - `Python 3.13`
- Предобработка данных - `numpy`, `pandas`
- Обучение регрессионных моделей для предсказания уровня успеваемости - `catboost` 
- Обучение моделей машинного обучения для нахождения аномалий - `scikit-learn`, `pyod`
- Интерпретация моделей -  `shap`
- Веб-интерфейс - `streamlit`

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <url>
   cd <repo>

2. Установите зависимости:
    ```bash
    pip install -r requirements.txt

3. Запустите приложение `learning.py`:
    ```bash
    streamlit run learning.py
    
4. Запустите приложение `detection.py`:
    ```bash
    streamlit run detection.py
